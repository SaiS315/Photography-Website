# Photography-Website
This is my website for my freelance photography gig. Quite simply built with HMTL, CSS, and Javascript. 

The site itself has four pages: an index/landing page, a portfolio gallery with different sections of work, a general privacy policy section
and a contact page where one can submit requests to work with me.

This website is now ready to use, and will continue to be edited for my own personal use and for users. I have just updated a few primary layout changes for mobile browser users and relocation of a few key areas for the website. The next issue I am aware of is the images taking a long time to load, so that is my next priority along with some text changes for visibility. 

Visit: [eyesofsai.netlify.app](https://eyesofsai.netlify.app/) to check it out, and check out my photography page on Instagram (@eyes_of_sai).
